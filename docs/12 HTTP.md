# 连接
每个客户端HTTP连接都将在以下阶段中运行。
- `ngx_event_accept()` 接受一个TCP客户端连接。这个处理器在响应的时候调用，以便读取监听套接字的端口。
在此阶段创建一个新的ngx_connecton_t对象。对象包裹新接受的客户端套接字。
每个nginx侦听器提供一个处理程序来传递新的连接对象。对于HTTP连接，是`ngx_http_init_connection(c)`。
